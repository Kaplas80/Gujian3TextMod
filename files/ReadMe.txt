Gujian 3 text mod version 0.2
Poject source was based on https://github.com/elishacloud/DirectX-Wrappers with modifcation. This has been modified and does not work the same as original source. Adapted the same license as in original project.
 
This wrapper DLL read game strings and replace them with a external file.
Support Steam .exe version 1.2.0.1890 and 1.3.0.2142

Change text in text.bin for 1.2.0.1890
Change text in text1302142.bin for 1.3.0.2142

Installation:
put dsound.dll and the text bin files to  game bin64/ directory alongside of gujian3.exe.

Uninstall:
remove above 3 files.
